# Vera Modeling Agency - Contest Platform

## About

Vera Modeling Agency is Scotland's premier modeling agency, dedicated to discovering and nurturing exceptional modeling talent. We are proud to present our exclusive modeling contest in partnership with an exciting new Scottish clothing brand.

## Contest Overview

We're hosting a modeling contest to find the face of a revolutionary new clothing brand that celebrates Scottish heritage with contemporary style. This is an incredible opportunity for aspiring and experienced models to launch or advance their careers.

### What Winners Receive:
- Exclusive modeling contract with the new clothing brand
- Professional photoshoot and portfolio development
- Representation by Vera Modeling Agency
- Feature in the brand's launch campaign
- Competitive compensation package
- Mentorship from industry professionals

### Eligibility:
- Age: 18-35 years old
- Height: 5'7" (170cm) or taller for female models, 5'11" (180cm) or taller for male models
- Available for photoshoots in Scotland
- No prior professional modeling experience required

## Website Features

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Website

1. Clone the repository:
```bash
git clone https://github.com/xxd232/Vera-.git
cd Vera-
```

2. Open the website:
   - Simply open `index.html` in your web browser
   - Or use a local web server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js with http-server
   npx http-server
   ```

3. Navigate to `http://localhost:8000` (if using a local server)
